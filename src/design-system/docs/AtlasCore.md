# Atlas Core

The Atlas Core is the canonical visual identity of Atlas.

It represents Atlas himself.

Every Atlas interface must use this component.

Examples:

Atlas Eye
Atlas Desktop
Atlas Nuclear
Atlas Legal
Atlas Medical
Atlas Robot

Atlas never changes identity.

Only operating state changes.
